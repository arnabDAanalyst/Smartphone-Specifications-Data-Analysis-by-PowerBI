# Smartphone Specifications Analysis

This dataset provides a comprehensive snapshot of the modern smartphone landscape, containing detailed specifications for nearly 1,000 unique smartphone models. 
It is designed to help researchers, analysts, and machine learning practitioners explore the relationships between hardware specs, brand value, and consumer pricing.

![Smartphone Specifications Data Analysis by PowerBI d-1](https://github.com/user-attachments/assets/3fb6e221-9bff-44d1-b31e-cc2ee5d3913e)

The dataset consists of 968 rows and 27 columns. Each row represents a specific smartphone model with its associated technical specifications and market price.
The data was collected from various mobile comparison and e-commerce websites. It has been cleaned to handle missing values and standardized for easy analysis.
